A Empresa (MEI)

Focamos no negócio de uma Confeitaria Artesanal em Salvador. O site foi projetado para ser um portfólio de produtos e um canal de orçamentos simples e direto.

html

HTML5 Semântico

Usamos tags como <header>, <main>, <section> e <footer> para garantir acessibilidade e melhor estrutura do documento.

css

CSS Moderno e Responsivo

Utilizamos Flexbox e CSS Grid para layouts fluidos, Variáveis CSS para o tema Dark e Media Queries para adaptação em dispositivos móveis (Mobile First).

js

JavaScript e DOM

Manipulação de DOM para o menu hambúrguer, Pop-up e alternância de temas. Validações completas de formulário com Regex.
